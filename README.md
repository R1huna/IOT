# Python for IoT – 수업 정리 레파지토리

이 레포지토리는 **IoT(Internet of Things) 프로그래밍 수업에서 배우는 파이썬 내용**을 체계적으로 정리하기 위해 만들어졌습니다.

파이썬 기초 문법부터 센서 활용, IoT 디바이스 제어까지 단계적으로 학습한 내용을 기록합니다.

---

## 📘 수업 목표

* 파이썬 기본 문법 이해
* IoT 환경에서 사용되는 파이썬 라이브러리 사용
* 센서 데이터 수집 및 처리
* 네트워크 통신 및 간단한 IoT 서비스 구현

---

## 📚 수업 진행 단계

### 1️⃣ 파이썬 기초

* 파이썬 설치 및 개발 환경 구성 (VS Code / IDE)
* 변수, 데이터 타입, 출력/입력
* 조건문(if), 반복문(for/while)
* 함수(function)
* 리스트, 딕셔너리 등 자료구조
* 모듈 & 패키지 사용법

### 2️⃣ 파이썬 심화

* 파일 입출력
* 예외 처리
* 클래스(OOP)
* 외부 라이브러리 사용하기 (pip)

### 3️⃣ IoT 기본

* IoT 개념 및 구조 이해
* 라즈베리파이/ESP 보드 등 개요
* GPIO 제어 기본
* 센서 데이터 읽기(온습도, 조도 등)

### 4️⃣ 네트워크 & 데이터 처리

* HTTP 요청 보내기
* MQTT 프로토콜 이해
* 클라우드에 데이터 전송하기
* 간단한 실시간 모니터링 구현
